단순한 모델 파라미터 조정이 아닌, **Human-in-the-loop 기반의 데이터 정제**를 통해 성능을 개선함. 특히 사용자 경험에 치명적인 도심 클래스 간 혼동을 획기적으로 줄여 상용화 가능한 수준의 신뢰성을 확보함.

**2. 클래스별 상세 성능 개선 분석**

**2.1 성능 급상승 클래스 (취약점 극복)**

초기 모델에서 0.8점대에 머물던 클래스들이 데이터 정제 후 0.9점대 이상으로 개선됨.

1. **Buildings (건물):** F1-Score 0.8846 → **0.9562**. 실제 건물을 놓치는 비율(Recall)이 0.87에서 0.95로 안정화됨.
2. **Street (거리):** F1-Score 0.8981 → **0.9625**. 재현율과 정밀도가 모두 0.96~0.97 수준으로 상승함.
3. **Parks (공원):** F1-Score 0.8766 → **0.9160**. 특히 정밀도(Precision)가 0.85에서 **0.94**로 폭등함.

**2.2 완벽에 가까운 클래스 (Top Performers)**

데이터 정제의 긍정적 영향으로 기존 우수 클래스들의 성능도 동반 상승함.

- **Person (인물):** Precision **1.0000(100%)** 달성.
- **Food (음식):** F1-Score **0.9913**으로 사실상 완벽 분류.
- **Animals (동물):** 정답 건수가 642건에서 653건으로 증가하며 F1-Score **0.9797** 기록.

**3. 오분류 패턴 개선 분석 (Confusion Matrix)**

의미적 유사성으로 인한 오분류(Semantic Ambiguity)가 데이터 정제 후 대폭 감소함.

- **도심 환경 (Buildings ↔ Street):**
    - Buildings → Street 오답: 24건 → **7건 (71% 감소)**.
    - Street → Buildings 오답: 18건 → **3건 (83% 감소)**.
    - **총 오분류 76% 감소**를 달성하여 도시 경관 분류의 경계를 명확히 함.
- **문화/자연 (Culture ↔ Parks):**
    - Culture → Parks 오답: 13건 → **4건 (69% 감소)**.
    - 배경(숲)이 아닌 피사체(건축물)에 집중하도록 정제한 결과, Culture의 정답 재현율이 437개에서 460개로 크게 상승함.
    - 다만, Parks → Culture 오답은 7건에서 10건으로 소폭 증가했으나 전체적인 신뢰도는 향상됨.

--------------------------------------------------------------------------------

**4. 시각적 검증 및 분석 (Grad-CAM)**

Grad-CAM 수행 결과, 모델이 배경이 아닌 객체 고유의 특징에 정확히 집중하고 있음을 확인함.

- **Buildings:** 도로가 아닌 **건물의 외벽과 창문**에 히트맵 활성화.
- **Street:** 건물이 아닌 **바닥(도로면)과 소실점**을 따라 히트맵 형성.
- **Culture:** 주변 숲이 아닌 **기와 지붕, 처마, 탑** 등 고유 양식에 집중.
- **Parks:** 인공물이 아닌 **나무, 잔디, 숲** 자체에 넓게 히트맵 분포.

--------------------------------------------------------------------------------

**5. 결론 및 향후 과제 (Final Insights)**

1. **데이터 품질의 중요성 입증:** 모델 튜닝 없이 오직 **데이터 정제만으로 F1-Score를 평균 0.03 이상 향상**시켜 데이터 중심 AI(Data-Centric AI)의 유효성을 증명함.
2. **서비스 신뢰성 확보:** 취약했던 도심지 데이터 간 혼동을 해결함으로써 '스마트 앨범' 서비스 상용화에 충분한 성능에 도달함.
3. **남은 과제 및 전략:**
    - 여전히 발생하는 Parks와 Culture 간의 혼동(10건)은 시각적 유사성 때문인 것으로 분석됨.
    - 다음 단계에서는 데이터 정제를 넘어 **Fine-tuning 및 Loss Weighting** 등 모델 학습 기법 고도화를 통해 미세한 차이를 학습할 예정임.
